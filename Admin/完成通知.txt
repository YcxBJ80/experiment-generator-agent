🚀 快速启动:
  1. cd Admin
  2. ./setup.sh  (或手动: npm install)
  3. cp .env.example .env.local
  4. 编辑 .env.local 填入 Supabase 配置
  5. npm run dev
  6. 访问 http://localhost:5174

📊 主要特性:
  • 用户统计卡片
  • 消息数量柱状图（Top 10）
  • 访问类型分布饼图

  • 一键分配访问类型

🔧 技术栈:
  • React 18 + TypeScript
  • Vite 5
  • Tailwind CSS
  • Recharts (图表)
  • Supabase

🔒 安全提示:
  • 不要提交 .env.local 到版本控制
  • 妥善保管 Service Role Key
  • 建议在内网环境使用
